# CPU Scheduling Algorithm Simulator 🖥️

A comprehensive Python-based simulator designed to calculate and visualize the performance of various CPU scheduling algorithms. This project provides an interactive GUI to help students and developers understand how Operating Systems manage process execution.

## ✨ Features
* **Supported Algorithms:**
    * First-Come, First-Served (FCFS)
    * Shortest Job First (SJF) - Preemptive & Non-Preemptive
    * Priority Scheduling
    * Round Robin (RR)
* **Real-time Calculations:** Automatically calculates Waiting Time (WT) and Turnaround Time (TAT).
* **Visual Representation:** Generates Gantt Charts for process execution flow.
* **User-Friendly GUI:** Built with Python's Tkinter/CustomTkinter library.

## 🛠️ Built With
* **Language:** Python
* **Libraries:** Tkinter (GUI), Matplotlib (Gantt Charting)
